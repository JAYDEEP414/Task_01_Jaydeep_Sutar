# Project 1: Data Cleaning & Preparation

## 📌 Objective
The goal of this project was to clean and prepare a raw e-commerce dataset for further analysis by handling missing values, duplicates, and formatting issues.

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- JupyterLab

---

## ✅ Tasks Performed
- Loaded the dataset using Pandas
- Checked missing/null values
- Handled missing data
- Removed duplicate records
- Verified duplicate Order IDs
- Standardized date formats
- Cleaned text formatting
- Validated TotalPrice calculations
- Exported cleaned dataset

---

## 📂 Files Included
- `Project_1.ipynb`
- `Cleaned_Dataset.csv`

---

## 📊 Skills Used
- Data Cleaning
- Data Preparation
- Data Validation
- Pandas Operations

---

## 🚀 Outcome
Successfully transformed a raw dataset into a clean and reliable dataset for analysis and future projects.
